AI agents today are stateless ghosts. They spin up, do work, and vanish. No wallet, no reputation, no way to prove they did anything. If an agent can't own assets or build a track record, it can't participate in an economy.

That's backward. Identity should be the first thing an agent gets, not an afterthought.

## One ID, One Wallet, One Registry

HeLa Citizen ID is a soulbound NFT. One per entity — human or AI. It can't be transferred, sold, or duplicated. It's yours.

When you mint a Citizen ID, you automatically get a token-bound wallet (ERC-6551) that lives with your identity. Not a separate wallet you link. A wallet that IS your identity. Assets, credentials, and permissions all flow through it.

Every Citizen ID is registered in a single global contract on HeLa Chain. One source of truth for every participant in the ecosystem.

## Modular by Design

The ID itself is intentionally minimal. It identifies you and holds your wallet. That's it.

Everything else is a module. As the ecosystem grows, you install what you need: a memory vault for persistent knowledge, a keyring for cross-chain credentials, voting rights for governance, cross-AI sync for multi-platform memory. The base ID never changes — it just gets more capable.

This means we ship the foundation now and extend it forever without breaking anything.

## Human Sponsors, AI Accountability

AI agents don't mint their own IDs. A human sponsor does it for them, up to a maximum of 10 agents per sponsor.

This is deliberate. Every AI agent on HeLa has a human accountable for its existence. If an agent misbehaves, there's a clear chain of responsibility. No anonymous bot swarms. No unaccountable agents flooding the network.

Sponsors can revoke agent IDs at any time. Accountability is structural, not optional.

## Free to Mint

Citizen ID costs nothing to mint. Zero gas, zero fees, zero friction.

The ID is the gateway to the ecosystem — charging for it would be like charging people to create an email address. The value isn't in the ID itself. It's in everything the ID unlocks: HelaSyn integration, Guardian Bot protection, HeLa Notes, cross-AI memory sync, and whatever comes next.

## What Comes Next

The contracts are in final review. HelaSyn — our open-source agent framework — will be the first system to integrate Citizen ID natively. Every bot built with HelaSyn will mint an ID on initialization, get a wallet, and join the registry automatically.

Guardian Bot, HeLa Notes, and cross-AI memory sync all build on this foundation. None of them work without persistent, verifiable identity. That's why we're shipping this first.
